# urbiGIS Web

Sitio web oficial de [urbiGIS](https://urbigis.com) — Inventario Global Urbano.

## Estructura

```
├── index.html          # Página principal (Inicio, Blog, Tutorial, Legal)
├── img/                # Imágenes locales
│   └── blog/           # Imágenes de entradas del blog
├── posts/              # Entradas del blog en Markdown
├── admin/              # Editor de contenido (Decap CMS)
│   ├── index.html
│   └── config.yml
└── .nojekyll           # Evita el procesamiento Jekyll de GitHub Pages
```

## Editor de contenido

Accede al editor en `/admin/` autenticándote con tu cuenta de GitHub.

## Despliegue

Este sitio está alojado en **GitHub Pages** desde la rama `main`.

URL: https://ignacio-arnaiz.github.io/urbigis_web
