Para registrar Accesos a Cuentas de Organización en urbiGIS, previamente el Administrador de dicha cuenta deberá facilitar al usuario interesado un Código de Cuenta. Este código de Cuenta viene adjunto siempre en el mail que urbiGIS envía a un usuario cuando se crea una Cuenta.

Una vez con el código de cuenta, se puede proceder a crear un nuevo acceso. Existen dos formas de registrar un acceso:

1. Solicitud a través del Módulo de Registro de Accesos

Sección titulada «1. Solicitud a través del Módulo de Registro de Accesos»

Se deben seguir estos pasos:

  • Paso 1. Solicitud: el solicitante abre el Panel de Módulos de Trabajo, selecciona Registro y luego Nuevo Acceso a Cuenta.

  • Paso 2. Formulario: se muestra el formulario de Registro de Accesos en el que el solicitante rellenará los campos necesarios. Los que están señalados con asterisco deben rellenarse obligatoriamente. Al hacer clic en Registrar se envía la solicitud para su tramitación por el personal de urbiGIS. El campo **Código de Cuenta **se completará con un código que será facilitado por el Administrador de dicha cuenta (que como se vio previamente, debe tener localizado).

  • Paso 3. Tramitación: La solicitud puede autorizarse o denegarse. Las solicitudes de registro de Accesos a una Cuenta concreta las tramitará su Administrador. Las solicitudes de Acceso genéricas que van acompañadas de alta de Cuenta las tramita el equipo de urbiGIS.

  • Paso 4. Alta: La autorización supone el alta del Acceso. Se envía una confirmación del alta por correo electrónico. A partir de este momento el Acceso ya puede iniciar sesión.

Sección titulada «2. Registro directo por el Administrador de una Cuenta»

Limitaciones

Sección titulada «Limitaciones»

  • Todo Acceso debe tener una dirección de correo válida y activa.

  • NO podrán registrarse dos Accesos distintos con la misma dirección de correo electrónico.

  • El mismo Acceso NO podrá pertenecer a dos Cuentas distintas

  • Sólo se puede solicitar el registro de Acceso perteneciendo a una Cuenta existente en urbiGIS. En este caso, el Administrador suministrará al solicitante el código de Cuenta para que éste lo introduzca en el formulario de solicitud.
